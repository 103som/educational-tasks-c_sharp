Общий список правил по работе с моей программой.
1) Ввод матрицы производится поэлементно строго через один пробел, без лишних символов в конце строки.
Пример корректного ввода для матрицы 3x3:
1 2 3
1 2 3
1 2 3
2) Есть возможность рандомно генерировать матрицу, при этом будет генерироваться матрица ранее введенного размера.(размер тоже можно генерить).
3) Если введенный размер не соотносится с введенной матрицей, то это ошибка и вводи заново.
4) Все элементы не превосходят по модулю 100.
5) Вводимая матрица не может быть больше 10 x 10.
6) Если при операции умножение/сложение/вычитание вы вввели некорректный размер одной из матриц, то операция не удастся и я предложу заново выбрать операцию(если вы решите продолжить работу программы).

Выше перечислил вроде все основные правила. P.S. Хорошей проверки,я старался!
